# YouTube Transcript Tool

A simple web application for fetching and copying YouTube video transcripts.

## Features

- Fetch transcripts from YouTube videos
- Copy transcripts with or without timestamps
- View recent searches
- Keyboard accessible

## Installation

```bash
pnpm install
```

## Development

```bash
pnpm dev
```
